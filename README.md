# Population Dashboard using Power BI

## Overview
This project is an interactive Power BI dashboard created to analyze global population trends and growth insights.

## Features
- Country-wise population analysis
- Year-wise population trends
- KPI cards for total population and growth %
- Interactive filters and slicers
- Map visualization

## Technologies Used
- Power BI
- Data Visualization
- DAX
- Excel

## Insights
- China and India have the highest population share
- Global population increased rapidly over the years
- Asia dominates population growth globally
